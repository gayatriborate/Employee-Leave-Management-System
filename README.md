# Employee-Leave-Management-System


## Project Description

Employee Leave Management System is a Django-based web application used to manage employee records and leave requests.

## Features

- Employee CRUD
- Leave CRUD
- Dashboard
- Search Employee
- Leave Filter
- REST API
- Bootstrap UI
- MySQL Database

## Technologies Used

- Python
- Django
- Django REST Framework
- MySQL
- HTML
- CSS
- Bootstrap 5
